## Pertanyaan Refleksi

1. Apa perbedaan [id].js dan [...slug].js?
   Jawab: [id].js adalah dynamic route biasa yg cuma bisa menangkap 1 url saja sedangkan [...slug].js adalah catch all route yg bisa menangkap
   beberapa url sekaligus

2. Mengapa slug berbentuk array?
   Jawab: karena slug itu bersifat catch all route, karena bisa menyimpan banyak url sekaligus maka slugnya berbentuk array

3. Kapan sebaiknya menggunakan Link dan router.push()?
   Jawab: link sebaiknya digunakan untuk tombol navigasi karena bibsa membuka halaman lebih cepat, dan untuk router.push() sebaiknya digunakan pada
   submit form atau login karena di  dalamnya masih ada proses pengecekan/verifikasi sebelum akhirnya diarahkan ke halaman

4. Mengapa navigasi Next.js tidak me-refresh halaman?
   Jawab: karena nextjs merefresh halaman di client side dan juga melakukan preload di background
