## Pertanyaan Refleksi

1. Apa fungsi utama _document.js?
   Jawab: fungsi _document.js adalah untuk mengatur struktur dasar, di praktikum awal disitu disuruh mengubah bahasa nya menjadi id berarti kita mengubah bahasa utama
   html nya dari english menjadi indonesia, itu sangat penting karena jika tidak diubah screen reader nya bisa salah membaca sebagai bahasa inggris seperti di beberapa web
   yang saya tahu bahasanya menjadi kacau

2. Mengapa <title> tidak disarankan di _document.js?
   Jawab: karena _document.js hanya dirender satu kali untuk membentuk struktur html, jika title dimasukkan kesitu maka tidak akan bbisa berubah saat kita  melakukan
   navigasi halaman

3. Apa perbedaan halaman biasa dan halaman 404.js?
   Jawab: halaman biasa memiliki route sesuai nama folder atau file, dan bisa diakses jika urlnya valid, sedangkan 404.js halaman khusus yg menampilkan error jika
   url atau route tidak ditemukan dan juga otomatis dipanggil

4. Mengapa folder public tidak perlu di-import?
   Jawab: karena folder public ini adalah folder bawaan nya nextjs jadi bisa langsung diakses lewat url dan file di dalamnya diangggap root path web nya
