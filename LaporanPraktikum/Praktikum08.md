   Server side rendering
   Side static generation

   mengambil data dari porduk, dan untuk SSR halaman akan dirender di server, tidak ada animasi skeletal loading seperti pada csr karena html yg dikirim server sudah
   ada data nya, tidak seperti csr yg memuat html kosong lalu browser merender datanya, sedangkan ssg dibuat saat kita sedang membuild web nya menggunakan npm run build dan
   halamannya bersifat statis

   Skeleton Loading
   Animasi
