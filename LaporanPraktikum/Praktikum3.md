## Pertanyaan Refleksi

1. apa perbedaan routing berbasis file dan routing manual
   Jawab: untuk routing berbasis file kita hanya perlu membuat file di folder pages dan file itu otomatis menjadi route, berbeda dengan routing manual
   dimana kita perlu mendeklarasikan routenya satu persatu

2. mengapa dynamic routinng itu penting dalam apllikasi web
   Jawab: karena jika kita menggunakan dynamic routing kita tidak perlu membuat banyak file , seperti pada tugas 2 dimana kita membuat blog
   menggunakan dynamic routing karena jika kita tidak memakai nya saat kita memiliki banyak artikel di blog tsb maka akan semqakin banyak juga file
   yang dibuat

3. apa keuntungan menggunakan layout global dibanding memanggil komponen satu per satu?
   Jawab: keuntungannya adalah kita tidak perlu membuat navbar ataupun footer di setiap halaman yang kita buat, karena dengan layout global navbar
   muncul di semua halaman jadi kita bisa menghemat waktu dan juga lebih cepat jika kita ingin mengubah tampilan global
