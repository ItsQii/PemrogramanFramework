## Pertanyaan Refleksi

1. Mengapa Pages Router disebut sebagai routing berbasis file?
   Jawab: karena url dibuat otomatis dari nama file jadi tidak perlu route manual

2. Apa perbedaan Next.js dengan React standar (CRA)?
   Jawab: Nextjs menggunakan SSR sedangkan React Standar (CRA) mmenggunakan CSR, lalu untuk routing nextjs menggunakan pages roouter berbasis file jadi     setiap file di folder pages otomatis jadi route sedangkan React routingnya manual menggunakan library

3. Apa fungsi perintah npm run dev?
   Jawab: fungsi nmpm run dev digunakan untuk menjalankan development server, untuk alamatnya yaitu localhost:3000, juuga saat kita mengubah kode kita perubahannya akan langsung ditampilkan tanpa perlu reload atau restart
   
4. Apa perbedaan npm run dev dan run build ?
   Jawab: npm run digunakan untuk membuat/mengembangkan aplikasi web nya sedangkan untuk run build digunakan saat kita sudah ingin mendeploy/hosting aplikasi yg kita buat
